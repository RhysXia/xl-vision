# 图标

语义化的矢量图形。图标来自[ionicons](http://ionicons.com/)

> **注意**

> 请使用`import { AddCircleOutline } from 'xl-vision/lib/icon`引入 icon。

> 引入 icon 时可能会导致包的体积变大，如果需要`tree-shaking`，减小体积。

> 请以这种方式引入:`import AddCircleOutline from 'xl-vision/lib/icon/AddCircleOutline`

# Icon API

| 参数      | 说明           | 类型                   | 可选值 | 默认值 | 是否必填 |
| --------- | -------------- | ---------------------- | ------ | ------ | -------- |
| clsPrefix | 自定义类名前缀 | string                 | -      | -      | false    |
| size      | 图标大小       | number &#x007c; string | -      | -      | false    |
| color     | 图标颜色       | string                 | -      | -      | false    |
| spin      | 图标是否旋转   | boolean                | false  | -      | false    |
| rotate    | 图标旋转角度   | number                 | -      | -      | false    |
