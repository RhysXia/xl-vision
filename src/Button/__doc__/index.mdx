# 按钮

基础的交互组件

:::demo 基础用法

通过`theme`、`variant`指定按钮的样式，按钮中也可以添加[Icon](#/icon)。

```jsx
import React from 'react'
import { Button } from 'xl-vision'
import { Search } from 'xl-vision/lib/icon'

export default () => {
  return (
    <div className='wrapper'>
      <div className='column'>
        <Button>Default</Button>
        <Button theme='primary'>Primary</Button>
        <Button theme='secondary'>Secondary</Button>
        <Button theme='error'>Error</Button>
        <Button theme='info'>Info</Button>
        <Button theme='warning'>Warning</Button>
        <Button theme='success'>Success</Button>
      </div>
      <div className='column'>
        <Button variant='outlined'>Default</Button>
        <Button variant='outlined' theme='primary'>
          Primary
        </Button>
        <Button variant='outlined' theme='secondary'>
          Secondary
        </Button>
        <Button variant='outlined' theme='error'>
          Error
        </Button>
        <Button variant='outlined' theme='info'>
          Info
        </Button>
        <Button variant='outlined' theme='warning'>
          Warning
        </Button>
        <Button variant='outlined' theme='success'>
          Success
        </Button>
      </div>
      <div className='column'>
        <Button variant='text'>Default</Button>
        <Button variant='text' theme='primary'>
          Primary
        </Button>
        <Button variant='text' theme='secondary'>
          Secondary
        </Button>
        <Button variant='text' theme='error'>
          Error
        </Button>
        <Button variant='text' theme='info'>
          Info
        </Button>
        <Button variant='text' theme='warning'>
          Warning
        </Button>
        <Button variant='text' theme='success'>
          Success
        </Button>
      </div>
    </div>
  )
}
```

:::demo 加载中

加载中的按钮不会响应用户操作，适合在某些需要延时完成的操作，可以避免用户的多次点击。

```jsx
import React from 'react'
import { Button } from 'xl-vision'
import { Trash, Close } from 'xl-vision/lib/icon'

const Loading = ({ children }) => {
  const [loading, setLoading] = React.useState(false)
  const timerRef = React.useRef()

  React.useEffect(() => {
    return () => {
      clearTimeout(timerRef.current)
    }
  }, [])

  const onClick = React.useCallback(() => {
    setLoading(true)
    timerRef.current = setTimeout(() => {
      setLoading(false)
    }, 3000)
  }, [])

  return React.cloneElement(children, {
    loading,
    onClick
  })
}

export default () => {
  return (
    <div className='wrapper'>
      <div className='column'>
        <Loading>
          <Button theme='primary'>Button</Button>
        </Loading>
        <Loading>
          <Button theme='primary' variant='outlined'>
            Button
          </Button>
        </Loading>
        <Loading>
          <Button theme='primary' variant='text'>
            Button
          </Button>
        </Loading>
      </div>
      <div className='column'>
        <Loading>
          <Button theme='primary' prefixIcon={<Trash />}>
            Button
          </Button>
        </Loading>
        <Loading>
          <Button theme='primary' variant='outlined' prefixIcon={<Trash />}>
            Button
          </Button>
        </Loading>
        <Loading>
          <Button theme='primary' variant='text' prefixIcon={<Trash />}>
            Button
          </Button>
        </Loading>
      </div>
      <div className='column'>
        <Loading>
          <Button theme='primary' suffixIcon={<Close />}>
            Button
          </Button>
        </Loading>
        <Loading>
          <Button theme='primary' variant='outlined' suffixIcon={<Close />}>
            Button
          </Button>
        </Loading>
        <Loading>
          <Button theme='primary' variant='text' suffixIcon={<Close />}>
            Button
          </Button>
        </Loading>
      </div>
    </div>
  )
}
```

```scss

  margin: 0.5rem;
}
```

## Button API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| clsPrefix | 自定义类名前缀 | string | - | - | false |
| href | 点击跳转地址，此时 button 行为与 a 标签一致 | string | - | - | false |
| disabled | 禁止按钮 | boolean | - | false | false |
| theme | 设置按钮主题 | string | default &#x007c; primary &#x007c; success &#x007c; warning &#x007c; error &#x007c; secondary | default | false |
| variant | 设置按钮类型 | string | contained &#x007c; outlined &#x007c; text | contained | false |
| long | 设置为长按钮 | boolean | - | false | false |
| prefixIcon | 前缀图标 | ReactElement | - | - | false |
| suffixIcon | 后缀图标 | ReactElement | - | - | false |
| loading | 设置按钮为加载中 | boolean | - | false | false |
| round | 圆角按钮 | boolean | - | - | false |
| elevation | 是否展示阴影效果，仅对`variant=contained` | boolean | - | - | false |

## Group API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| clsPrefix | 自定义类名前缀 | string | - | - | false |
| round | 指定按钮组中按钮的圆角样式 | boolean | - | false | false |
| vertical | 指定按钮组为垂直方向 | boolean | - | false | false |
| size | 设置按钮大小,如果按钮自动设置了 size，则会覆盖这个值 | string | default &#x007c; large &#x007c; small | default | false |
