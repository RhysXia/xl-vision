# 按钮

基础的交互组件

:::demo 基础用法

通过`theme`、`variant`指定按钮的样式，按钮中也可以添加[Icon](#/icon)。

```jsx
import React from 'react'
import { Button } from 'xl-vision'
import { Search } from 'xl-vision/lib/icon'

export default () => {
  return (
    <div className='wrapper'>
      <div className='column'>
        <Button>Default</Button>
        <Button theme='primary'>Primary</Button>
        <Button theme='secondary'>Secondary</Button>
        <Button theme='error'>Error</Button>
        <Button theme='info'>Info</Button>
        <Button theme='warning'>Warning</Button>
        <Button theme='success'>Success</Button>
        <Button theme='primary' href='#'>
          Link
        </Button>
      </div>
      <div className='column'>
        <Button variant='outlined'>Default</Button>
        <Button variant='outlined' theme='primary'>
          Primary
        </Button>
        <Button variant='outlined' theme='secondary'>
          Secondary
        </Button>
        <Button variant='outlined' theme='error'>
          Error
        </Button>
        <Button variant='outlined' theme='info'>
          Info
        </Button>
        <Button variant='outlined' theme='warning'>
          Warning
        </Button>
        <Button variant='outlined' theme='success'>
          Success
        </Button>
        <Button variant='outlined' theme='primary' href='#'>
          Link
        </Button>
      </div>
      <div className='column'>
        <Button variant='text'>Default</Button>
        <Button variant='text' theme='primary'>
          Primary
        </Button>
        <Button variant='text' theme='secondary'>
          Secondary
        </Button>
        <Button variant='text' theme='error'>
          Error
        </Button>
        <Button variant='text' theme='info'>
          Info
        </Button>
        <Button variant='text' theme='warning'>
          Warning
        </Button>
        <Button variant='text' theme='success'>
          Success
        </Button>
        <Button variant='text' theme='primary' href='#'>
          Link
        </Button>
      </div>
    </div>
  )
}
```

```scss
.column::deep .xl-button {
  margin: 0.5rem;
}
```

## Button API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| clsPrefix | 自定义类名前缀 | string | - | - | false |
| dashed | 设置按钮边框为虚线样式，需要和`plain`连用才能显示出效果 | boolean | - | false | false |
| disabled | 禁止按钮 | boolean | - | false | false |
| ghost | 幽灵属性，使按钮背景透明 | boolean | - | false | false |
| href | 点击跳转地址，此时 button 行为与 a 标签一致 | string | - | - | false |
| loading | 设置按钮为加载中 | boolean | - | false | false |
| long | 设置为长按钮 | boolean | - | false | false |
| plain | 设置为朴素按钮 | boolean | - | false | false |
| shape | 设置按钮的形状，通常`circle`与[Icon](#/icon)连用 | string | cirlce &#x007c; round | - | false |
| theme | 设置按钮类型 | string | default &#x007c; primary &#x007c; success &#x007c; warning &#x007c; error &#x007c; secondary | default | false |
| size | 设置按钮大小 | string | default &#x007c; large &#x007c; small | default | false |
| prefixIcon | 前缀图标 | ReactElement | - | - | false |
| suffixIcon | 后缀图标 | ReactElement | - | - | false |

## Group API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| clsPrefix | 自定义类名前缀 | string | - | - | false |
| round | 指定按钮组中按钮的圆角样式 | boolean | - | false | false |
| vertical | 指定按钮组为垂直方向 | boolean | - | false | false |
| size | 设置按钮大小,如果按钮自动设置了 size，则会覆盖这个值 | string | default &#x007c; large &#x007c; small | default | false |
