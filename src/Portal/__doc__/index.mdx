# Portal

可以灵活方便地将子组件挂载到其他位置的组件

:::demo 基础用法

基础用法

```jsx
import React from 'react'
import { Portal, Button } from 'xl-vision'

export default () => {
  const [node, setNode] = React.useState()

  React.useEffect(() => {
    const portal = (
      <Portal getContainer='.container2'>
        <Button theme='primary' className='content'>
          点击
        </Button>
      </Portal>
    )
    setNode(portal)
  }, [])

  const onClick = React.useCallback(() => {
    alert('You clicked container1!')
  }, [])

  return (
    <div>
      <div className='container1' onClick={onClick}>
        <p>container1</p>
        {node}
      </div>
      <div className='container2'>
        <p>container2</p>
      </div>
    </div>
  )
}
```

```scss
.container1,
.container2 {
  height: 100px;
  color: #fff;
  text-align: center;
  padding: 15px;
  border-radius: 4px;

  > p {
    font-size: 20px;
    font-weight: bold;
  }
}

.container1 {
  background-color: #8bc34a;
}

.container2 {
  margin-top: 16px;
  background-color: #e91e63;
}
```

## Portal API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| getContainer | 指定挂载的父节点，支持字符串，节点实例或返回这些的函数 | string &#x007c; HTMLElement &#x007c; (() => string &#x007c; HTMLElement) | - | - | false |
| children | 子元素 | ReactNode | - | — | true |
