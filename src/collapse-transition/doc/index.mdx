
# CollapseTransition

:::demo 基础用法

基础用法

```jsx
import {CollapseTransition, Button} from '@xl-vision/xl-vision'
export default () => {
     const [active, setActive] = React.useState(false)

     return (
       <div className='demo'>
         <Button onClick={() => {setActive(!active)}}>Click</Button>
         <CollapseTransition
           show={active}
           transitionClassName={'demo-fade'}
         >
           <div className='demo-box'>DEMO</div>
         </CollapseTransition>
         <style jsx>{`
            .demo {
              &-box {
                height: 10rem;
                color: #fff;
                font-size: 3rem;
                line-height: 10rem;
                text-align: center;
                vertical-align: middle;
                background-color: #007bff;
              }

              :global(&-fade) {
                transition: height 2s ease;
              }
            }
        `}</style>
       </div>
     )
}

```

:::

## CollapseTransition API

| 参数   | 说明                                       | 类型            | 可选值 | 默认值 | 是否必填|
| ------ | ------------------------------------------ | --------------- | ------ | ------ | ---|
| show   | 触发进出场动画,`true`表示进场，`false`表示出场 | boolean | -   | —      | true|
| transitionClassName | 指定动画过程中的class名,如果不指定，会使用默认动画`height 1s ease`            | string  | -  | -|false|
| children   | 子元素            | React.ReactElement | -   | —      |true|
