# Tooltip

::: demo 基本用法

基本的使用方式

```jsx
import { Tooltip } from 'xl-vision'
export default () => {
  return (
    <Tooltip content={'tooltip text'}>
      <span>Tooltip will show on mouse enter</span>
    </Tooltip>
  )
}
```

::: demo 支持不同的触发方式

通过`trigger`指定触发方式

```jsx
import { Tooltip, Button } from 'xl-vision'
export default () => {
  const [visible, setVisible] = React.useState(false)
  return (
    <Button.Group>
      <Tooltip trigger='hover' content={'hover text'}>
        <Button>hover</Button>
      </Tooltip>
      <Tooltip trigger='click' content={'click text'}>
        <Button>click</Button>
      </Tooltip>
      <Tooltip trigger='focus' content={'focus text'}>
        <Button>focus</Button>
      </Tooltip>
      <Tooltip trigger='contextMenu' content={'contextMenu text'}>
        <Button>contextMenu</Button>
      </Tooltip>
      <Tooltip trigger='custom' visible={visible} content={'custom text'}>
        <Button onClick={() => setVisible(!visible)}>custom(click twice)</Button>
      </Tooltip>
    </Button.Group>
  )
}
```

::: demo 指定弹出框挂载的位置

通过`getPopupContainer`可以指定弹出框挂载的位置

```jsx
import { Tooltip, Button } from 'xl-vision'
export default () => {
  return (
      <Tooltip trigger='hover' content={'hover text'} getPopupContainer={null}>
        <Button>hover</Button>
      </Tooltip>
  )
}
```

## Tooltip API

| 参数      | 说明                     | 类型            | 可选值 | 默认值 | 是否必填 |
| --------- | ------------------------ | --------------- | ------ | ------ | -------- |
| prefixCls | 自定义类名前缀           | string          | -      | -      | false    |
| content   | 提示内容,覆盖 popup 属性 | React.ReactNode | -      | -      | true     |

## Popup 公共 API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| lazyRender | 延时挂载弹出框 | boolean | - | - | true |
| visible | 手动设置 visible | boolean | - | - | false |
| trigger | 触发方式 | 'hover' &#x007c; 'focus' &#x007c; 'click' &#x007c; 'contextMenu' &#x007c; 'custom' | - | 'hover' | false |
| placement | 弹出框位置 | 'top' &#x007c; 'left' &#x007c; 'right' &#x007c; 'bottom' &#x007c; 'top-start' &#x007c; 'top-end'&#x007c; 'bottom-start' &#x007c; 'bottom-end' &#x007c; 'left-start' &#x007c; 'left-end' &#x007c; 'right-start' &#x007c; 'right-end' | - | 'top' | false |
| overlayStyle | 弹出框样式 | React.CSSProperties &#x007c; ((placement: Placement) => React.CSSProperties) | - | - | false |
| overlayClassName | 弹出框 className | string &#x007c; ((placement: Placement) => string) | - | - | false |
| onVisibleChange | 弹出框弹出或隐藏事件 | (visible: boolean) => void | - | - | false |
| getPopupContainer | 弹出框挂载的位置,`null`表示挂载在当前 dom | HTMLElement &#x007c; Selectors &#x007c; null &#x007c; (() => HTMLElement &#x007c; Selectors &#x007c; null) | - | () => document.body | false |
| delayShow | 延迟显示 | number | - | 0 | false |
| delayHide | 延迟隐藏 | number | - | 0 | false |
| allowPopupEnter | 是否允许进入提示框 | boolean | - | true | false |
| children | 参考元素 | React.ReactElement | - | - | true |
| arrow | 箭头 | (placement: Placement, center: { x: number, y: number }) => React.ReactElement | - | - | false |
| offset | 弹出框距离参考元素的距离 | number | - | - | false |
| onVisibleChange | 显示状态改变时的回调 | (visible: boolean) => void | - | - | false |
| popup | 弹出框 | (placement: Placement) => React.ReactNode | - | - | true |
| transitionName | 显示隐藏时的动画 className | CssTransitionClassNames &#x007c; ((placement: Placement) => CssTransitionClassNames) | - | - | false |
| popperModifiers | 自定义 popper.js 的 modifiers 配置项，具体参见[popper.js 文档](https://popper.js.org/popper-documentation.html#modifiers) | Modifiers | - | - | { preventOverflow: { boundariesElement: 'window' } } | false |
