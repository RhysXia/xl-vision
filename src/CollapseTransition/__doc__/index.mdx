# CollapseTransition

可以方便地对于不定高度或者宽度的内容进行折叠或者展开的动画组件。

:::demo 基础用法

通过`CollaseTrannsition`包裹对应的元素来触发折叠和展开的动画效果。通过`transitionClassName`自定义动画方式。如果不指定，将使用默认动画`width/height 1s ease`

```jsx
import React from 'react'
import { CollapseTransition, Button } from 'xl-vision'
export default () => {
  const [active, setActive] = React.useState(false)

  return (
    <div className='wrapper'>
      <Button
        onClick={() => {
          setActive(!active)
        }}
      >
        Click
      </Button>
      <CollapseTransition in={active} transitionClassName={'demo-fade'}>
        <div className='demo'>DEMO</div>
      </CollapseTransition>
    </div>
  )
}
```

```scss
.demo {
  height: 10rem;
  color: #fff;
  font-size: 3rem;
  line-height: 10rem;
  text-align: center;
  vertical-align: middle;
  background-color: #007bff;
}

.wrapper::deep .demo-fade {
  transition: height 2s ease;
}
```

:::

::: demo 水平模式

通过属性`horizontal`在水平方向进行展开和折叠动画

export default () => {

## CollapseTransition API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| in | 触发进出场动画,`true`表示进场，`false`表示出场 | boolean | - | — | true |
| transitionClassName | 指定动画过程中的 class 名,如果不指定，会使用默认动画`height(width) 1s ease` | string | - | - | false |
| children | 子元素 | React.ReactElement | - | — | true |
| horizontal | 是否横向 | boolean | - | — | - |
| transitionOnFirst | 是否初始化完成时就触发动画 | boolean | - | false | false |
| mountOnEnter | 当第一次进场动画时才挂载子节点，见<Link to="Transition">**Transition**</Link> | boolean | - | false | false |
| unmountOnLeave | 当出场动画后就卸载子节点，见<Link to="Transition">**Transition**</Link> | boolean | - | false | false |
