# BaseButton

基础的 button 组件，只提供了基础的功能，可以方便使用者自定义。

## 代码演示

:::demo 自定义按钮

自定义按钮

```jsx
import React from 'react'
import { BaseButton } from 'xl-vision'

export default () => {
  return (
    <div className='container'>
      <BaseButton className='custom'>custom css</BaseButton>
    </div>
  )
}
```

```scss
@use '../style/var' as *;

.container {
  &::deep {
    .custom {
      padding: 6px 16px;
      border-radius: 3px;
      background-color: #9c27b0;
      color: #fff;

      @include button;
      @include elevation(2);

      &:focus {
        opacity: 0.85;
      }
    }
  }
}
```

:::demo 复杂按钮

复杂按钮

```jsx
import React from 'react'
import { BaseButton } from 'xl-vision'
import Breakfast from './breakfast.jpg'
import Burgers from './burgers.jpg'
import Camera from './camera.jpg'
const images = [
  {
    url: Breakfast,
    title: 'Breakfast',
    width: '40%'
  },
  {
    url: Burgers,
    title: 'Burgers',
    width: '30%'
  },
  {
    url: Camera,
    title: 'Camera',
    width: '30%'
  }
]

export default () => {
  return (
    <div className='container'>
      {images.map((it, index) => (
        <BaseButton
          key={index}
          style={{
            width: it.width
          }}
          className='button'
        >
          <span
            className='img'
            style={{
              backgroundImage: `url(${it.url})`
            }}
          />
          <span className='title'>{it.title}</span>
        </BaseButton>
      ))}
    </div>
  )
}
```

```scss
.container {
  &::deep {
    .button {
      position: relative;
      height: 200px;
      color: #000;
      &:focus {
        opacity: 0.85;
      }
    }
    .img {
      display: block;
      position: absolute;
      left: 0;
      top: 0;
      right: 0;
      bottom: 0;
      background-size: cover;
    }
    .title {
      position: absolute;
      left: 50%;
      top: 50%;
      transform: translate(-50%, -50%);
      color: #fff;
      font-size: 20px;
      font-weight: bold;
    }
  }
}
```

## BaseButton API

| 参数          | 说明               | 类型      | 可选值 | 默认值 | 是否必填 |
| ------------- | ------------------ | --------- | ------ | ------ | -------- |
| clsPrefix     | 自定义类名前缀     | string    | -      | -      | false    |
| disableRipple | 是否禁止水波纹特效 | boolean   | -      | true   | false    |
| disabled      | 是否开启禁止状态   | boolean   | -      | false  | false    |
| loading       | 是否开启加载状态   | boolean   | -      | false  | false    |
| href          | 指定点击跳转的链接 | string    | -      | -      | false    |
| children      | 子节点             | ReactNode | -      | -      | true     |
