# 图标

语义化的矢量图形。图标来自[ionicons](http://ionicons.com/)

> **注意**

> 请使用`import { AddCircleOutline } from 'xl-vision/lib/icon`引入icon。

> 引入icon时可能会导致包的体积变大，如果需要`tree-shaking`，减小体积。

> 请以这种方式引入:`import AddCircleOutline from 'xl-vision/lib/icon/AddCircleOutline`

::: demo 基础用法

基本使用方法

```jsx
import React from 'react'
import { AddCircleOutline, Alarm, Albums, ArrowBack, Reload } from 'xl-vision/lib/icon'
export default () => {
  return (
    <div className='demo'>
      <AddCircleOutline size='2rem' />
      <Alarm size={32} />
      <Albums size={32} color='red' />
      <ArrowBack size={32} rotate={20} />
      <Reload size={32} spin />
    </div>
  )
}
```

```scss
.demo::deep .xl-icon {
  margin-right: 1rem;
}
```

:::

# Icon API

| 参数      | 说明           | 类型                   | 可选值 | 默认值 | 是否必填 |
| --------- | -------------- | ---------------------- | ------ | ------ | -------- |
| clsPrefix | 自定义类名前缀 | string                 | -      | -      | false    |
| size      | 图标大小       | number &#x007c; string | -      | -      | false    |
| color     | 图标颜色       | string                 | -      | -      | false    |
| spin      | 图标是否旋转   | boolean                | false  | -      | false    |
| rotate    | 图标旋转角度   | number                 | -      | -      | false    |
